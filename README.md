### `Nvm (Node version management)`

To enable/install the nvm package follow this instruction:
Open terminal, go to link below, if you already have it type: $ nvm use:

Follow the instructions on this link:
[Installing Nvm using brew](https://formulae.brew.sh/formula/nvm)

Once installed, restart your sessions/terminal and type:
$ nvm --version
output should be something like:
0.35.2

To install a specific version that the project is using:
$ nvm install >=11.10.1

If your project have a .nvmrc file. If not, add it and simple type only the version needed for the project.
To use it:
$ nvm use
